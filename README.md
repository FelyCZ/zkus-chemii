# Zkus chemii
Rozcestník pro středoškolské studenty pro jednoduchý přehled o mimoškolních soutěžích, 
soutředěních a jiných akcích se zaměřením na chemii.

## Použité technologie
* Nuxt
* Nuxt UI
* Tailwind
* Nuxt Image
